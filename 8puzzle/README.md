使用A*算法解决数码问题，其中要使用之前学过的堆或者说优先队列结构，A* * 算法是一种启发式搜索算法，specification推荐使用两种优先级函数：汉明距离和曼哈顿距离，前者是计算当前board和目标board之间相同位置不同的count，后者是计算当前board每个块与目标board对应距离的横纵坐标差之和（两者都不把空白快计算在内

A*算法大致就是首先把初始状态入队，然后出队最接近目标board的node（表现为距离最小），判断是否是目标board，若是，则终止，若不是，则将其neighbor如队，neighbor是指当前board移动一步之后所能达到的board，直到目标board

但是注意到，并不是每个初始board都是可解的，如果不可解，仅仅按照上面的步骤，就会死循环，此时有一种办法，实现在不可解时，及时退出循环：将初始board随意两块交换（不包括空白块即0），得到twinBoard，他和初始board只有依着是可解的，所以在循环中，如果twinBoard可解，那么即可知初始board不可解，循环终止。

另一点，在获取neighbor时，会获取到上一步，也就可能会反复移动，为了避免这一点，在把neighbor加入到优先队列时，首先判断是否是当前board的上一步，若不是，则加入。
